# Portfolio Feature Wishlist

Here are the fun and interactive features we've planned. We'll tackle them one by one.

## Feature Queue

- [x] **Project Rating / Like Button with Animations**: Visitors can "like" a project (persisted in `localStorage`), and an animated heart will pop.
- [ ] **Animated Skill Bars + Confetti at 100%**: Animate skill bars into view on scroll, and trigger a confetti burst when a skill hits 100%.
- [ ] **Hover Sound Toggle & Volume Control**: Add a UI toggle to enable/disable the existing hover sound, and include a volume slider.
- [x] **3D Tilt Cards with Shine Effect**: Make project cards tilt in 3D based on the user's cursor position, with a glossy shine effect that follows the pointer.
- [x] **Persona Chatbot / Quick Intro**: Implement a small, non-intrusive chat bubble that provides playful introductions or navigational hints.
- [ ] **Language Toggle (EN / HI)**: Add a button to switch site content between English and Hindi, using fun and playful translations.
- [ ] **Scroll Progress Bar**: A sleek progress bar at the top of the viewport that fills up as the user scrolls down the page.
- [x] **Konami Code Easter Egg**: Typing the classic Konami Code (↑ ↑ ↓ ↓ ← → ← → B A) triggers a fun surprise (like fireworks or a theme shift).
- [ ] **Custom Right-Click Menu**: A stylized context menu that replaces the browser default, offering quick links to projects or social media.
- [ ] **Page Transition Animations**: Implement smooth fade-in/out transitions when navigating between different HTML pages.
- [ ] **Custom Theme Color Picker**: Allow visitors to customize the site's accent color (beyond just Light/Dark mode) to their preference.
- [ ] **Live GitHub Activity Feed**: Fetch and display the latest public commits or activity from GitHub using the GitHub API.
- [x] **Real-time Weather Widget**: Display the current weather in your location (e.g., "Coding from Mumbai 🌧️") in the footer or about section.
- [ ] **Interactive 404 Page**: Create a custom "Page Not Found" page with a mini-game or fun animation for lost visitors.
- [ ] **Spotlight Cursor Effect**: A "flashlight" mode where the screen dims and a light follows the cursor to reveal content.
- [ ] **Keyboard Shortcuts**: Add power-user navigation (e.g., press 'P' for Projects, 'C' for Contact, '?' for help).
- [ ] **Print-Friendly CSS**: Optimize the layout so that printing the page (Ctrl+P) generates a clean, resume-style document.
- [ ] **Lazy Load Blur-up**: Implement a performance feature where images load with a blurry placeholder that sharpens into focus.
- [x] **Mini Typing Speed Test**: A small, interactive game component to test typing speed, relevant for a developer portfolio.
